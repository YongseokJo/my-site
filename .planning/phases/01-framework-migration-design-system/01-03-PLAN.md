   - Import `@retorquere/bibtex-parser` for BibTeX parsing
   - Import `js-yaml` for YAML output
   - Import `fs` and `path` from Node.js stdlib
   - Implement these functions matching the Python script:
     a. `cleanLatex(s)` -- Remove BibTeX braces `{}`, convert LaTeX accents to Unicode (same accent map as Python script: `\\'a` -> `a`, `\\"o` -> `o`, `\\~n` -> `n`, `\\c{c}` -> `c`, `\\ss` -> `ss`, etc.). Use `String.prototype.normalize("NFC")` for final normalization.

     c. `normalizeJournal(journal)` -- Map LaTeX journal macros to abbreviations: `\\apj` -> `ApJ`, `\\mnras` -> `MNRAS`, `\\aap` -> `A&A`, `\\aj` -> `AJ`, `\\apjs` -> `ApJS`, `\\apjl` -> `ApJL`. Strip braces.
