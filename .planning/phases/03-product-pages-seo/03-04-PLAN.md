Per D-13 (NOT Netlify Forms) and UI-SPEC BetaSignupForm interaction contract:

```typescript
import { useState } from "react";
import { Input } from "@/components/ui/input";
import { Label } from "@/components/ui/label";
import { Button } from "@/components/ui/button";

interface FormData {
  name: string;
  email: string;
}

interface FormErrors {
  name?: string;
  email?: string;
}

type FormStatus = "idle" | "submitting" | "success" | "error";

// Use the EXACT validation messages from UI-SPEC copywriting contract
function validate(field: keyof FormData, value: string): string | undefined {
  if (field === "name" && !value.trim()) return "Name is required.";
  if (field === "email" && !/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(value))
    return "Please enter a valid email address.";
  return undefined;
}
```

Form state management: identical pattern to ContactForm.tsx (useState for formData, errors, status; handleBlur, handleChange, handleSubmit).

```typescript
const response = await fetch("https://formspree.io/f/YOUR_BETA_FORM_ID", {
  method: "POST",
  headers: { "Content-Type": "application/json", Accept: "application/json" },
  body: JSON.stringify({ name: formData.name, email: formData.email }),
});
```
Use `YOUR_BETA_FORM_ID` as placeholder — user will replace with real Formspree form ID.
